# FloodGame
Console Flood Game in C++

How to Compile:
  Notes
  - the make file is include for easy compiling  
  - output binary is set to "output"

  How to compile
  - open terminal
  - run the make command from this directory using: "make"

to run game, run the binary file from terminal:
   - ./output
